# RAG

Prompt Engineering

RAG（Retrieval-Augmented Generation）是一种结合检索和生成的自然语言处理模型，旨在提高生成模型的性能和效率。RAG模型通过检索大型文本数据库中的信息，以辅助生成模型生成更准确、更连贯的文本输出。

(找知识、查资料)

场景：

- 问答系统：RAG可以用于构建高效的问答系统，通过检索相关文档来回答用户的问题。

进化 => Fine-tuning（增量学习训练：阅读、背诵、理解资料）
